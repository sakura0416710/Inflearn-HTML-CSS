input type속성들

input type을 제대로 해야 스마트폰에서 최적의 자판이 보이게 되므로 text를 남발하지 말자.

form 요소 : 사용자로부터 데이터(정보)를 입력받기 위한 HTML요소이다. 입력받은 정보를 form단위로 한꺼번에 전송함

하지만 최근에는 사용하지 않는 추세이다. 싱글페이지app에서는 알맞지 않기 때문이다.

form태그의 여러가지 요소

당신이 배우고 싶은 언어는? (다중 선택 가능)

하나의 그룹을 나타내는 도구 : name="" 통일시키기.
for=""설정을 해야 글씨 눌러도 체크가능

당신이 배우고 싶은 언어는? (하나만 선택 가능)

name을 통일시켜야 선택지 중 하나만 선택가능함(같은 라디오체크 그룹이라고 인식하기 때문)
for=""설정해야 글씨 눌러도 체크가 가능하다 (input이랑 label이랑 연결해주니까)

input에서 색상팔레트 띄우기 : type : color

캘린더 띄우기 : input type:date

시간까지 선택 : type="datetime-local

이메일 : type="email"

하지만 요즘은 디자인된 걸로 자동검사를 하기 때문에 기본타입을 사용하지는 않는다.

파일첨부 type="file"

hidden : 안보이지만 정보입력 가능한 속성. 사용자의 입력을 받기 위한 용도가 아니라,
개발자가 사용하기 위해서 쓰는 태그이다. 숨겨진 값도 함께 전송하기 위해 쓰임
ex. value값으로 현재 주문하는 사람의 정보를 함께 넘겨야 누가 주문했는지 알 수 있다.

input 이미지 속성 : 잘 안쓰기 함

month타입

주문 수량 등에 쓸 수 있는 number type

비밀번호 password type

전화번호 : tel type

form태그를 써서 자동으로 형식을 체크해줄 수 있지만 각국마다 전화번호 형식이 다르므로 개발자가 지정해주어야 한다.
따라서 pattern이라는 속성으로 (자바스크립트) 체크할 수 있다.

그외 모든 문자, 숫자를 받을 수 있는 type="text"

시간만 선택하는 type= time

type="url"
