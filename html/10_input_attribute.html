input의 속성값

1. readonly : 읽기전용 (값 전송가능)

2. disabled : 수정불가 (값 전송 불가)

3. maxlength : 입력 자릿수 설정

4. placeholder : 입력 가이드 (클릭 시 사라짐)

5. required : 필수 입력값

autofocus : 입력할 칸으로 자동이동

6. autocomplete : 캐쉬에 저장된 정보가 보이게 됨 (사이트 상관없이 쌓임)

name값이 중요한 역할을 한다. 표준name값으로 맞춰줄 것
브라우저에 저장되는 정보는 name속성의 값에 영향을 받고 자동완성 기능을 사용할 수 있는 미리 정의된 name값이 있다.
1. name : 사람 전체이름
2. email : 이메일 주소
3. username: 사용자 계정
4. new-password :새로운 비번
5. current-password : 사용자의 현재 비번
6. one-time-code : 사용자를 인증할 떄 사용하는 1회성 코드
7. organization-title : 직위
8. organization : 회사 또는 조직명
9. country : 국가코드
10. country-name : 국가이름
11. postal-code : 우편번호
12. address-level1 : 첫번째 행정구역 (우리나라는 특별시.광역시.도)
13. address-level2 : 두번째 행정구역 (우리나라는 시.구)
14. address-level3 : 세번째 행정구역
15. address-level4 : 네번째 행정구역
16. street-address : 도로 주소
17. language : 선호 언어
18. bday : 생년월일
19. bday-day : 생년월일에서 일
20. sex : 성별
21. tel : 국가코드 포함한 전체 전화번호
22. tel-country-code : 국가코드
23. tel-national : 전체전번에서 국가코드 제외한 전번
24. tel-area-code : 지역번호
25. tel-local : 국가코드랑 지역번호 제외한 전화번호
26. url: 웹사이트 주소
