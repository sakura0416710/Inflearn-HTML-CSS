flexbox 레이아웃

flexbox는 웹 화면의 공간분할과 콘텐츠가 담길 공간의 크기 및 배분, 그리고 콘텐츠에 대한 배치 순서를 빠르고 쉽게 정의할 수 있게 해주는 화면 레이아웃 모델. 주축 main axis과 교차축 cross axis에 기준으로 레이아웃을 정의할 수 있음. row: 아이템이 배치되는 방향이 왼 -> 오른쪽 row-reverse : 아이템 배치 방향 오른 ->왼 column : 상단 -> 하단 column-reverse : 하단->상단
